periodLabel

periodLabel: AccessibleLabel
  • Defined in types/events/basketball.ts:98

Presentable label describing the current period. This will offer a text description of the period - e.g., HT (half time) or 1st Qtr (first quarter)

In non-MidEvent statuses, this will offer a text description of the status based on the next/previous most appropriate period. E.g., in PreEvent status, this field will return 'Scheduled'. In PostEvent, it will return a label related to the period after which the match ended - e.g., 'FT', 'FT OT'
