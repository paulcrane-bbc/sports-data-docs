periodLabel

periodLabel: AccessibleLabel
  • Defined in src/types/events/football.ts:99

Presentable label describing the current period.

When period is FootballPeriod.FirstHalf, FootballPeriod.SecondHalf, FootballPeriod.ExtraTimeFirstHalf or FootballPeriod.ExtraTimeSecondHalf this value will describe the current match clock time in the format M' (e.g., 27') and the accessible value will be a pronouncable equivalent (e.g., 27 minutes)

In other periods, this will offer a text description of the period - e.g., HT (half time)

In non-MidEvent statuses, this will offer a text description of the status based on the next/previous most appropriate period. E.g., in PreEvent status, this field will return 'Scheduled'. In PostEvent, it will return a label related to the period after which the match ended - e.g., 'FT', 'AET', 'PENS'
