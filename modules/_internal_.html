BaseQuery

BaseQuery: { type: string; urn: string }
  • Defined in types/queries.ts:10

Type declaration

  • type: string

    Type of query. See each subtype for possible values.

  • urn: string

    URN referencing the sports team, event, tournament, round or stage.

    In the format of: urn:bbc:sportsdata:SPORT:PART:ID

    Where:

    SPORT is a sport name such as football, basketball etc

    PART can be team, event, tournament, round or stage

    ID is the identifier within the part (e.g., a team slug)

    example

    urn:bbc:sportsdata:football:team:liverpool
